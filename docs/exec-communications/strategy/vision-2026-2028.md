# Vision 2026–2028

## North Star
Become the default intelligent caching layer for every edge-deployed application globally.

## 2026 Goals
- GA launch of CPM Runtime v1.0 (Cloudflare Workers, Fastly Compute)
- 28 paying customers, $6.8M ARR
- SOC 2 Type II certification
- Federated Learning v1.0 shipped

## 2027 Goals
- LLM-augmented cache reasoning (CPM v3.0)
- 110 customers, $22M ARR
- Third-party model marketplace launched
- ISO 27001 certification

## 2028 Goals
- Cross-edge federated intelligence (anonymized cross-customer learning)
- Cache-as-a-Service (CaaS) managed offering
- 340 customers, $58M ARR
- Evaluate Series B / path to profitability

## Strategic Pillars
1. **Edge-first ML** — inference at the node, not the cloud
2. **Privacy by design** — federated learning, no raw data centralization
3. **Platform agnostic** — integrates with any CDN or edge runtime
4. **Data flywheel** — more deployments → more signal → better models → more customers
